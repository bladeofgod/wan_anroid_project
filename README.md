# wan-android
抄了不少flutter项目代码了，这次仿照玩安卓界面自己写一个。

此项目中加了大量注释，以及将源码中举的例子翻译成了中文，以方便阅读。
非常适合初学者使用。

PS: WanAndroid 中的登录和收藏API不太好使，相应页面没法返回具体结果。
